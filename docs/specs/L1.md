# Get Well App — L1 Requirements

## L1-1: Welcome / Onboarding Screen
The app shall present a welcome screen that introduces Get Well, explains its mission, and allows the user to choose between "Get Well" (general support) and "Get Well Faith" (Christian-values-based support) modes.

## L1-2: Chat Interface — Get Well Mode
The app shall provide a warm, empathetic, judgment-free AI chat interface for general emotional support, available 24/7 with anonymous session handling.

## L1-3: Chat Interface — Get Well Faith Mode
The app shall provide a Christian-values-based AI chat interface with prayer prompts and scripture references for faith-based emotional support.

## L1-4: Crisis Detection & Resource Referral
The app shall detect crisis signals in conversation and present appropriate crisis resources (hotlines, text lines) with clear, compassionate routing.

## L1-5: Donation Screen
The app shall provide a donation interface that allows supporters to contribute to the 501(c)(3) nonprofit, with full transparency and no pressure.

## L1-6: Settings / About Screen
The app shall provide a settings and about screen with information about the nonprofit mission, privacy assurance, and app configuration.

## L1-7: React Native App Infrastructure
The app shall be built as a React Native application with typed navigation, a shared design system (theme, colors, typography), and reusable components matching the design specifications.

## L1-8: End-to-End Tests (Detox + Screenplay Pattern)
The app shall have comprehensive Detox E2E tests organized using the Screenplay pattern (Actors, Abilities, Tasks, Questions, Interactions) covering all user flows.
