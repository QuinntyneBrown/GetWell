# Contributing to Get Well

Thank you for your interest in contributing to Get Well. This project exists to provide free, anonymous emotional support to anyone who needs it, and every contribution moves that mission forward.

## How to Contribute

### Reporting Issues

- Open an issue describing the bug, enhancement, or question
- Include steps to reproduce for bugs
- Include screenshots or screen recordings when relevant

### Submitting Changes

1. Fork the repository
2. Create a feature branch from `main` (`git checkout -b feature/your-feature`)
3. Make your changes
4. Write or update tests as appropriate
5. Ensure all tests pass
6. Commit with a clear, descriptive message
7. Push to your fork and open a pull request against `main`

### Pull Request Guidelines

- Keep PRs focused on a single change
- Reference any related issues in the PR description
- Include a summary of what changed and why
- Add screenshots for any UI changes
- Ensure your code follows the existing style and conventions

## Development Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Follow the platform-specific setup for React Native (iOS/Android)
4. Run the app locally to verify your changes

## Areas Where Help Is Needed

- **Mobile development** — iOS and Android (React Native)
- **AI integration** — Chat interface and API integration
- **Anonymous session handling** — Secure, stateless session management
- **Donation processing** — Payment integration for 501(c)(3) contributions
- **Crisis detection** — Improving detection accuracy and resource routing
- **Accessibility** — Ensuring the app is usable by everyone
- **Localization** — Translating the app for non-English speakers

## Code of Conduct

- Be kind and respectful in all interactions
- Remember that this project serves people in their most vulnerable moments
- Prioritize user safety and privacy in every decision
- No personal data should ever be collected or stored
